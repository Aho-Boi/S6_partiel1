# Bugs

## Decrire un bug

Voici ce qui caracterise un bug:

* Severity / Priority
* La date de creation du bug
* Comment produire le bug
* Sur quelle platforme il apparait.
* Que fait le bug
* Le resultat attendu
* Le responsable
* Cout de resolution
* La version du programme
* Title + Unique ID
* Reproductibility rate
* Status
